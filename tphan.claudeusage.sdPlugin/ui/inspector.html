Reads the OAuth token Claude Code stores in your macOS Keychain (Claude Code-credentials) and calls Anthropic's usage endpoint — no login here. The Keychain prompt names the security tool; choose Always Allow (not just "Allow") so it doesn't re-ask. Press the key any time to refresh now. The usage endpoint is rate-limited, so keep the interval at 60s or higher.
A dimmed key is showing the last-good reading because the last poll failed. Press it to retry. With no reading yet, the key shows a hint: keychain = grant access (Always Allow); sign in = sign in with Claude Code; setup = offline / rate-limited.
